# Tailwind CSS 

## Tailwind CSS 개념
> https://tailwindcss.com/

`HTML을 떠나지 않고 빠르게 빌드하는 모던 웹사이트`

CSS를 별도로 작성하지 않아도, HTML 파일에서의 마크업 만으로 웹페이지 구축 가능 <br>
=> utility-first 기법 (미리 구현된 CSS 사용)

<br><br>

## 장점
1. HTML 문서만 보면 된다. 
2. class 이름을 고민할 필요가 없다.
3. 유지보수에 용이하다.
